# Magicards-Online
In this card game : Fight to the death your opponents until you are the last one standing !

## [v1.25b+] Complete overhaul of gaming experience for both gamemodes

- You now play directly on the game interface ! To choose a card move left or right with the arrow key then press enter. If the card has multiple uses choose one with the same arrows then press enter again. Finally, if the card is to be used on an adversary use the arrow keys to chose which player to attack then press enter. And voilà ! Way simpler than before.
- Furthermore the picking and discard actions have been made very simple : To pick a new deck press (p) to discard a card go to it with the arrows then press (d). To finish your turn press (f)
